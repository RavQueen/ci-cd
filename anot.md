comandos do git:
git branch -> mostra as branchs do repositório
git branch {nome} -> cria uma branch (sem retorno)
git switch {nome} -> troca o branch ativo
git branch -D {nome} -> apaga a branch
git status -> mostra os status da branch, o que foi enviado ou não
git add {nome} -> adiciona na lista de espera (sem retorno) "." equivale a tudo
git commit -m "" -> mostra os commit né
git push origin {nome da branch} -> envia tudo na lista de espera pra branch
